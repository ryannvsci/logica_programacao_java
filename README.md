# Lógica de Programação com Java
Este repositório contém exercícios de lógica de programação implementados em Java.


## Conteúdo

- Exercícios básicos de entrada e saída de dados
- Estruturas condicionais e loops
- Manipulação de variáveis e operações matemáticas
- Exercícios com arrays, cálculos e lógica aplicada

## Exercícios incluidos

- Exercício 1: Soma A + B e comparação com C
- Exercício 2: Verificar se número é par/ímpar e positivo/negativo
- Exercício 3: Somar ou multiplicar A e B conforme igualdade
- Exercício 4: Imprimir antecessor e sucessor de um número
- Exercício 5: Calcular quantos salários mínimos o usuário ganha
- Exercício 6: Aplicar reajuste de 5% em um valor
- Exercício 7: Verificar se dois valores booleanos são ambos verdadeiros ou falsos
- Exercício 8: Ordenar três valores inteiros em ordem decrescente
- Exercício 9: Calcular média de três notas
- Exercício 10: Calcular média de quatro notas e determinar aprovação
- Exercício 11: Calcular valor final do produto conforme forma de pagamento
- Exercício 12: Verificar maioridade
- Exercício 13: Trocar valores entre variáveis
- Exercício 14: Calcular tempo vivido (anos, meses e dias)
- Exercício 15: Identificar tipo de triângulo
- Exercício 16: Conversão Fahrenheit para Celsius
- Exercício 17: Comparar crescimento de altura entre duas pessoas
- Exercício 18: Tabuada fixa do 9
- Exercício 19: Tabuada personalizada
- Exercício 20: Gerar número aleatório entre 0 e 100
- Exercício 21: Divisão com quociente e resto
- Exercício 22: Cálculo de salário líquido
- Exercício 23: Cálculo de litros consumidos em viagem
